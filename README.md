# Filtered Logging Proxy

## Features

- bare-server
- logging
- hash-chain logs
- QuickTunnel
- webhook URL reporting
- skips:
  - mp4
  - images
  - wasm
  - css
  - js
  - html
  - pdf

## Install

```bash
npm install
```

## Download cloudflared

```bash
curl -L \
https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64 \
-o cloudflared

chmod +x cloudflared
```

## Start

```bash
npm start
```

## Environment Variables

WEBHOOK_URL=https://discord.com/api/webhooks/xxxxx
